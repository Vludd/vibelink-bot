## VibeLink Bot

**VibeLink Bot** - Telegram-бот для поиска людей на одной волне по интересам, целям и формату общения.

Проект помогает пользователям находить друзей, тиммейтов для игр, людей для совместных проектов, бизнес-контакты или просто собеседников с похожими увлечениями. Пользователь создает профиль, указывает свои интересы и цель поиска, после чего бот показывает подходящие анкеты других людей.

Основная механика построена на взаимном интересе: пользователь может поставить лайк понравившейся анкете, а контакт собеседника открывается только после взаимного лайка. Такой подход помогает снизить спам, сохранить приватность и сделать общение более комфортным.

### Основные возможности

* создание пользовательского профиля;
* выбор цели поиска: дружба, общение, игры, проекты, бизнес или отношения;
* подбор людей по интересам и совпадениям;
* карточки пользователей с краткой информацией;
* механика лайков и взаимных коннектов;
* открытие Telegram username только после взаимного интереса;
* готовые подсказки для первого сообщения;
* настройки приватности и безопасности.

### Идея проекта

VibeLink - это не просто бот для знакомств. Его цель - помочь человеку найти подходящего собеседника под конкретный контекст: поиграть вечером, обсудить идею, собрать команду, найти единомышленников или начать общение с человеком со схожими интересами.

Главный фокус проекта - быстрый, понятный и безопасный UX внутри Telegram.
